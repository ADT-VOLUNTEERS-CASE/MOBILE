## Что делает приложение

- [x] Помогает волонтерам находить и записываться на мероприятия по интересам
- [x] Обеспечивает инструменты управления для координаторов и администраторов
- [ ] Ведет персональную статистику участия: количество мероприятий, отработанные часы, роли
- [ ] Формирует публичный рейтинг волонтеров

## Цели проекта

1. Уйти от ручного учета, исключить ошибки при вводе данных и сократить время на формирование отчетности.
2. Дать волонтерам и организаторам доступ к актуальной информации в любой момент.
3. Обеспечить круглосуточный доступ с мобильных устройств независимо от местоположения.

## Технологии

- **Язык:** Kotlin
- **Платформа:** Android (API 33+, compileSdk 36)
- **UI:** Jetpack Compose
- **Архитектура:** Clean Architecture с разделением на модули || MVVM в presentation слое
- **DI:** Hilt / Dagger
- **Сеть:** Retrofit + Kotlinx Serialization
- **Асинхронность:** Kotlin Coroutines
- **Хранение:** DataStore Preferences (локальные настройки)
- **Тестирование:** JUnit 5, MockK, Roborazzi, ArchUnit

## Структура модулей

```text
presentation/   Экраны, ViewModel, навигация, темы и компоненты UI
domain/         Бизнес-логика и контракты репозиториев
data/           Работа с сетью, репозитории данных, модели API
core/           Общие инструменты, тестовые фикстуры
storage/        Локальное хранение данных через DataStore
```

## Роли пользователей

Приложение поддерживает три роли с разными правами доступа:

- **Волонтер** — поиск и запись на мероприятия
- **Координатор** — управление событиями и участниками
- **Администратор** — управление доступом

## Основной функционал

### Раздел «Мероприятия»

- Карточки с обложкой, названием, датой и адресом
- Текстовый поиск

### Экран мероприятия

- Полная информация о событии
- Отображение доступных мест
- Кнопка записи или отмены участия

### Раздел «Профиль»

- Личная информация и контакты
- Интересы в виде тегов
- Редактирование профиля
- Выход из аккаунта

## Требования для сборки

- Android Studio с поддержкой Kotlin 2.2+
- JDK 21
- Gradle (версия управляется wrapper)

## Сборка

```bash
./gradlew assembleDebug
```

Для подписи релизной сборки необходимы переменные окружения:
- `ANDROID_KEYSTORE_PATH`
- `ANDROID_KEYSTORE_PASSWORD`
- `ANDROID_KEY_ALIAS`
- `ANDROID_KEY_PASSWORD`
